```

Entering rescue mode ...
grub rescue> _
```

## grub rescue修复

命令：
```
grub rescue>ls
grub rescue>set
```


## ubuntu18.04引导修复

修复工具：`boot-repair`

通过ubuntu的U盘安装进入试用模式，并进行联网

```
sudo add-apt-repository ppa:yannubuntu/boot-repair
sudo apt-get update
```
```
sudo apt-get install -y boot-repair
boot-repair
```
根据提示选择yes，直到修复成功。
